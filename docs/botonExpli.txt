Sistema de Personalización de Botón Tragalero
============================================

Este archivo (boton.html) contiene ÚNICAMENTE el código necesario para integrar el sistema de personalización de botones en cualquier otra página o plataforma.

Instrucciones de Integración:
----------------------------

1. Dependencia Externa:
Para que el botón funcione (el proceso de pedido real), debes incluir el script de GloriaFood en tu página principal:
<script src="https://www.fbgcdn.com/embedder/js/ewm2.js"></script>

2. Cómo usar este código:
Copia todo el contenido de 'boton.html' y pégalo directamente en el lugar de tu HTML donde quieras que aparezca el personalizador.

3. Qué incluye:
- Estilos CSS (encapsulados en clases únicas para evitar conflictos).
- Estructura HTML (Inputs de CUID, RUID y Colores).
- Lógica JS del Componente: Define la etiqueta <tragalero-orders>.
- Lógica de Sincronización: Actualiza la vista previa y genera el código final.

Atributos del Botón:
--------------------
Si solo quieres usar el botón final sin el personalizador, usa esta estructura:

<tragalero-orders
  cuid="TU_CUID"
  ruid="TU_RUID"
  background="#COLOR_HEX"
  color="#COLOR_TEXTO_HEX">
</tragalero-orders>

Nota: Asegúrate de que el script de GloriaFood (paso 1) esté cargado en la página para que el botón sea funcional al hacer clic.
